# Portfolio - Thomas NOGARET

Ce site est le portfolio numérique de **Thomas Nogaret**, étudiant en 3ᵉ année de BUT Mesures Physiques à l'IUT de Bordeaux.

## ⚙️ Technologie
Site statique réalisé en HTML5 et CSS3, conçu pour un hébergement GitHub Pages.

## 📁 Structure
- `index.html` : contenu principal du site
- `style.css` : style graphique (bleu IUT Bordeaux)
- `CV-NOGARET-Thomas.pdf` : CV téléchargeable
- `photo.jpg` : portrait
- `hero.jpg` : image de bannière
- `README.md` : documentation

## 🌐 Déploiement GitHub Pages
1. Crée un dépôt sur ton GitHub, par exemple : `portfolio-thomas-nogaret`
2. Téléverse tous les fichiers.
3. Dans les paramètres du dépôt :
   - onglet **Pages**
   - sélectionne la branche `main`, dossier `/root`
   - clique **Save**
4. Ton site sera accessible à l’adresse :  
   👉 `https://<ton-github-username>.github.io/portfolio-thomas-nogaret/`
